This project is an early prototype showing how a chatbot experience can be built with a large language model. The original page embedded the app directly, but the site now links to it externally to improve loading speed and mobile usability.

## Project summary

The prototype explores how conversational AI can help users interact with information in a more natural way. Instead of clicking through many menus, users can ask questions directly and receive generated responses.

## What this project demonstrates

- Interest in applied AI, not just theory
- Ability to turn model experimentation into a user-facing concept
- A practical bridge between machine learning and product experience
